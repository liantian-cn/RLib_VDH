## 使用说明

下载此插件，安装到插件目录。

下载[RLib](https://github.com/liantian-cn/RLib)，安装到插件目录。

在设置菜单内，启动PixelUI

按说明运行  [RLib_PixelRunner](https://github.com/liantian-cn/RLib_PixelRunner)


## 常用宏


```pl

#showtooltip
/cast  [@cursor] 沉默咒符
/delay 0.4


#showtooltip
/cast  [@cursor]  悲苦咒符
/delay 0.4


#showtooltip
/use  [@cursor]  烈焰咒符
/delay 0.4
/cast 恶魔尖刺


#showtooltip
/use  [@cursor]  怨念咒符
/delay 0.4


#showtooltip
/use 恶魔追击
/delay 0.4


```
